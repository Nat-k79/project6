# Проект домашнее задание

## Описание проекта

Проект позволяет получать информацию о банковских операциях с картами клиентов.
Для модулей masks, widget и processing созданы тесты.

## Структура проекта
- **src/**: папка содержит основные функции прокта
- **tests/**: папка содержит тесты для модулей

## Установка
### для пользователя
Для установки зависимостей (без разработки) используйте
'''
poetry install --no-dev
'''

### для разработчика
Для установки всех зависимостей (включая разработку) используйте
'''
poetry install
'''

## Примеры использования
Перед запуском проекта необходимо установить зависимости. Для этого выполните
'''
poetry install
'''
