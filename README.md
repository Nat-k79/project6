# Проект домашнее задание

## Описание проекта

Проект позволяет получать информацию о банковских операциях с картами клиентов.
Для модулей masks, widget, processing и generators созданы тесты.
Создан декоратор log. Он будет автоматически регистрировать детали выполнения функций:
- время вызова функции
- имя функции
- передаваемые аргументы
- результат выполнения функции
- информацию об ошибках, если они возникают

Это обеспечит более глубокий контроль и анализ поведения программы в процессе ее выполнения 

## Структура проекта
- **src/**: папка содержит основные функции проекта
- **tests/**: папка содержит тесты для модулей

## Установка
### для пользователя
Для установки зависимостей (без разработки) используйте
'''
poetry install --no-dev
'''

### для разработчика
Для установки всех зависимостей (включая разработку) используйте
'''
poetry install
'''

## Примеры использования
Перед запуском проекта необходимо установить зависимости. Для этого выполните
'''
poetry install
'''

## Тестирование
Ко всем функциям проекта написаны тесты. Модули тестируются на корректность работы

### Структура тестов

Тесты находятся в директории tests/. Каждый файл теста соответствует определенному модулю проекта и содержит набор тестов для проверки его функциональности

## Модуль генераторов

Модуль содержит функции-генераторы для обработки данных транзакций
